Let
  1. M presents the number of division;
  2. x presents the ratio of height hidden above align position;
  3. y presents the corresponding percent of background-position required;

Then
    M * HEIGHT_UNIT * y - HEIGHT_UNIT * y = M * HEIGHT_UNIT * x
                (M - 1) * HEIGHT_UNIT * y = M * HEIGHT_UNIT * x
                                        y = ( M / ( M - 1) ) * x
So	 
     x <= (0 .. M-1) / M
     y => (0 .. M-1) / (M - 1)
